Este programa traduce los ficheros .txt localizados en la carpeta "Input",
a 105 idiomas distintos. Cada una de las traducciones se encontrara localizada
dentro de subcarpetas dentro de la carpeta denominada "Output"

Si una linea, del contenido que se quiere traducir, comienza con "#", se entendera
que es un comentario y se omitira su traduccion. Asimismo se eliminara la # en el fichero traducido que se origine
