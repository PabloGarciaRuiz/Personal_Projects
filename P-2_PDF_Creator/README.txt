Este proyecto genera un PDF a partir de un fichero de texto .txt

+ Para el texto del documento:
	- Si se escribe algo del estilo:
		"casa: house"
	  Se va a poner en negrita a "casa:", ya que se entendera que se trata de una
	  definicion, y por lo tanto se desearia un cambio entre las distintas partes

+ Se permite escribir lineas separadoras:
	- "---" introducira una linea separadora sin espacios adicionales
	- "----"  con un espacio adicional
	- "-----"  con dos espacios adicionales 
	- "------"  con tres espacios adicionales 

+ Se permite dibujar cajitas:
	- Se permite dibujar una cajita vacia usando "[]"
	- Se permite dibujar una cajita con texto usando "[]<-texto"
